# Compare Library APIs

1. Resolve both libraries or versions independently in Context7.
2. Query equivalent features and use the same comparison dimensions for both sides.
3. Compare documented API shape, lifecycle, error handling, extension points, and compatibility.
4. Use a concise table for exact mappings, followed by meaningful tradeoffs.
5. Mark missing documentation as unknown rather than treating absence as lack of support.
6. Recommend one only after relating the tradeoff to the user's stated constraints.
