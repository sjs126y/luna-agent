# Operate Web Page

1. Navigate to the explicit user target and inspect the current page before acting.
2. Fill or select reversible inputs as requested. Never infer secrets or personal data.
3. Before final submission, purchase, publication, deletion, external message, or account change, summarize the exact effect and obtain explicit confirmation.
4. File upload and arbitrary page-code execution remain unavailable unless plugin configuration enables them.
5. After an approved action, verify the resulting page state and report what actually happened.
