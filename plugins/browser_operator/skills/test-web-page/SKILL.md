# Test Web Page

1. Restate the target URL, flow, expected result, and any allowed test credentials.
2. Inspect the page before acting, then perform the smallest deterministic sequence.
3. Verify state after every meaningful action; capture screenshot or console/network evidence for failures.
4. Never perform purchases, destructive account actions, real external messages, or production mutations.
5. Report steps, expected result, actual result, evidence, and a concise reproduction path.
